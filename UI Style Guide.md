# DigiCal – Admin UI Style Guide

---

## Global Look & Feel
- Primary color: **#2271b1** (WordPress admin blue)
- Buttons:
  - Primary: Blue background / white text / hover scale 1.05
  - Delete: Red background / white text
- Forms: compact height, light grey border, 4–8px spacing

---

## Tables
- Responsive, auto-fit; wrap before 80% width
- Sticky header (white text / blue background)
- First column: checkbox selection for bulk actions
- Hover row highlight
- Primary rows: **bold text**
- Child rows: slight indent

---

## Sections
- Collapsible containers:
  - Start collapsed by default
  - Header click toggles expand/collapse
- Expand All / Collapse All button globally
- Section header counts:
  - True total row count (not only visible rows)

---

## Modal-Free Editing
- **Inline editing** for all modules:
  - Click a cell → activates edit mode
  - Enter to save
- No popups where not needed

---

## Icons & Labels
- Configure button: small, green, white text
- Role chips (speakers): badge styling (coming in v1.2)

---

## Accessibility
- Keyboard navigation enabled
- ARIA-expanded on toggle buttons
